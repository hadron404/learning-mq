
== docker 安装 rabbitmq

1.拉取镜像
----
docker pull rabbitmq:management
----

2.启动容器，运行 rabbitmq
----
docker run -d -p 5672:5672 -p 15672:15672 --name rabbitmq-management rabbitmq:management
----

3.访问 http://+{ip}+:{port}/

此处port = 15672, ip= 127.0.0.1
默认账号密码： guest/guest
